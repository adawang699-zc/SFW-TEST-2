代理程序方式（推荐）

代理程序运行在远程主机上，提供精确的报文构造和发送能力。

  1. Windows 安装步骤：
    • 安装 Npcap: https://nmap.org/npcap/
    • 安装 Python 3.7+
    • 安装依赖: pip install flask flask-cors scapy
    • 运行代理程序: python packet_agent.py --host 0.0.0.0 --port 8888
    • 注意：需要管理员权限运行
  2. Linux 安装步骤：
    • 安装依赖: sudo apt-get install python3-pip libpcap-dev
    • 安装Python包: pip3 install flask flask-cors scapy
    • 运行代理程序（需要root）: sudo python3 packet_agent.py --host 0.0.0.0 --port 8888
  3. 连接信息：
    • 代理程序URL：例如 http://192.168.1.100:8888
    • 确保代理程序已启动并监听指定端口
    • 确保防火墙允许访问代理程序端口
SSH方式（备用）

通过SSH连接远程主机，使用系统命令发送报文（功能有限）。

  1. Windows 7/10/11 需要先安装并启用 SSH 服务器：
    • Windows 10/11: 设置 → 应用 → 可选功能 → 安装"OpenSSH 服务器"
    • Windows 7: 下载安装 OpenSSH for Windows
    • 启动服务: Start-Service sshd
  2. 连接信息：
    • 主机地址：远程主机的 IP 地址
    • SSH端口：默认 22
    • 用户名和密码：系统用户凭据

推荐：使用代理程序方式可以获得更精确的报文控制和更好的性能。
