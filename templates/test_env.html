常见命令说明

ipconfig
ipconfig /all - 查看详细信息
查看所有网络接口的IP配置信息

netsh interface ip add address "以太网" 192.168.1.100 255.255.255.0
为指定网卡添加IP地址（不删除原有IP）

netsh interface ip set address "以太网" static 192.168.1.100 255.255.255.0 192.168.1.1
设置静态IP地址（会覆盖原有IP），最后一个是网关

netsh interface ip delete address "以太网" 192.168.1.100
删除指定网卡的指定IP地址

route print
route print -4 - 仅显示IPv4路由
route print -6 - 仅显示IPv6路由
显示系统路由表

route add 192.168.2.0 mask 255.255.255.0 192.168.1.1
route add 192.168.2.0 mask 255.255.255.0 192.168.1.1 metric 1 if 2
添加路由：目标网络 子网掩码 网关 [metric 跃点数] [if 接口索引]

route delete 192.168.2.0
删除指定目标网络的路由

arp -a
arp -a -N 192.168.1.1 - 查看指定接口的ARP表
显示ARP缓存表

netstat -an - 显示所有连接和监听端口
netstat -ano - 显示进程ID
netstat -ano | findstr :8888 - 查看指定端口
显示网络连接和监听端口信息

ip addr show
ifconfig
ip addr show eth0 - 查看指定网卡
查看所有网络接口的IP配置信息

ip addr add 192.168.1.100/24 dev eth0
ifconfig eth0:0 192.168.1.100 netmask 255.255.255.0
为指定网卡添加IP地址（不删除原有IP）

ip addr flush dev eth0 && ip addr add 192.168.1.100/24 dev eth0
ifconfig eth0 192.168.1.100 netmask 255.255.255.0
设置IP地址（会覆盖原有IP）

ip addr del 192.168.1.100/24 dev eth0
ifconfig eth0:0 down - 删除别名IP
删除指定网卡的指定IP地址

ip route show
route -n
netstat -rn
显示系统路由表

ip route add 192.168.2.0/24 via 192.168.1.1 dev eth0
route add -net 192.168.2.0/24 gw 192.168.1.1 dev eth0
添加路由：目标网络 via 网关 dev 接口

ip route del 192.168.2.0/24
route del -net 192.168.2.0/24
删除指定目标网络的路由

ip neigh show
arp -a
arp -n - 不解析主机名
显示ARP缓存表

netstat -tlnp - 显示TCP监听端口和进程
ss -tlnp - 使用ss命令（推荐）
lsof -i :8888 - 查看指定端口
netstat -tlnp | grep :8888 - 过滤指定端口
显示网络连接和监听端口信息
